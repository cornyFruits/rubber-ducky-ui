info

what

a rubber ducky is a 'bad usb' that mimics a keyboard and automatically types pre-programmed keystrokes when plugged into a computer.

because it looks like a regular, harmless usb, it bypasses many security measures. it can be used maliciously, but is also used for penetration testing.

how

rubber duckies work by emulating a human interface device (hid), like a keyboard. once plugged in, they execute scripts in duckyscript, a scripting language.

these scripts can perform any action that a human typing on a keyboard can do such as opening a terminal, configuring settings and downloading malicious software.

security implications

rubber ducky attacks highlight the risks of physical access and the trust placed in usb devices. it is easy to use and deploy, which makes it a significant threat in all environments.

these devices: can execute malicious commands, are suspectible to network attacks, exploit the trust os systems have in usb devices and can be used to steal sensitive information.

hospitality attack

In 2020, hackers targeted hotel chains using rubber ducky devices. employees unknowingly plugged these devices into company infrastructure, which triggered malicious scripts that exfiltrated guest data and installed backdoors into the company's network.

the attack compromised sensitive customer information, such as payment details, demonstrating how these devices exploit human vulnerbilities.

stuxnet attack

this was not technically a rubber ducky attack, but it did rely on usb drives to infiltrate iranian nuclear facilities in 2010.

usb drives, containing malicious payloads, targeted specific industrial control systems. this attack highlighted how damaging usb-based methods can be and how physical access can be used with cyber-espionage.

prevention

organisations can mitigate rubber ducky attacks by: educating employees on the risks and dangers of plugging in unknown drives into devices, disabling usb ports when necessary, implementing endpoint protection software and keeping systems and firmware updated - to name a few.
